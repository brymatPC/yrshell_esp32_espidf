# YRShell ESP32 ESP-IDF

This project is based on work done by ssanci in the repository https://github.com/salsanci/YRShell.  The primary goals:

1. Provide a mechanism for using YRShell with esp-idf
1. Remove arduino espressif layer
1. Provide another example of how YRShell can be used on a project
1. Add in the BLE functionality of the ESP32 IC

# Setup Hardware
This library has been tested on the ESP32.

# Setup PC Software
This library is configured for use with esp-idf within vscode.
